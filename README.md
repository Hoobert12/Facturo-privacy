# Politique de confidentialité — Facturo

Dernière mise à jour : mars 2026

## Aucune collecte de données

Facturo ne collecte, ne transmet, ni ne partage aucune donnée personnelle.

Toutes les données saisies dans l'application (informations entreprise, 
clients, factures) sont stockées exclusivement sur votre appareil, 
dans le stockage local iOS.

## Aucun serveur distant

L'application ne communique avec aucun serveur. Aucune donnée ne quitte 
votre appareil.

## Sauvegardes

Les fichiers de sauvegarde (.facturxbackup) sont créés localement et 
partagés uniquement à votre initiative, via les outils iOS natifs 
(AirDrop, Mail, Fichiers). Facturo n'a aucun accès à ces fichiers 
une fois partagés.
